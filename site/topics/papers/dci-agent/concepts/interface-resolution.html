    Think of it like…

    Reading an entire book vs jumping to the exact sentence on page 12. Higher resolution = smaller, more precise units of access.

    Retrieval Interface Resolution

    TL;DR: "Resolution" is how fine-grained the agent's corpus access is — document, passage, line, or character span.

    Why this matters

    The paper argues retrieval quality depends on interface granularity — what the agent can observe, verify, and act upon — not just retriever embedding quality.

    Resolution slider

    Drag from document → span. Feel what "higher interface resolution" means.

    Mechanism

    DCI exposes tool outputs: matched lines with line numbers, file paths, surrounding context from targeted reads. Retriever-mediated access typically caps at passage-level snippets.
