    Unix pipes: combine weak clues (`grep warranty | grep SKU-8842`) instead of one semantic guess that might miss exact terms.

    CLI Composability

    TL;DR: DCI's power is composable shell primitives — grep, find, read, piped together by the agent.

    Why this matters

    Multi-hop agentic search often needs sparse clue conjunctions and exact string matches — hard to express as a single embedding query.
